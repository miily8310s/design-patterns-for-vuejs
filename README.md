## Vue Testing Library 　のインストール

[公式ドキュメント](https://testing-library.com/docs/vue-testing-library/intro)に従ってインストール

```bash
vue create app-name

# manual installを選択
# vue3/typescript/unit testを最低選択

yarn add -D @testing-library/vue
```
